# 100DaysOfCodeInIoT Challenge Day 25

Go Serverless Generate Presigned URL for S3 file upload.

## Requirements
- AWS Account
- AWS CLI
- Go

## Building
- Install serverless: `yarn install`
- Run `make build`

## Deploying
Run `make deploy`
